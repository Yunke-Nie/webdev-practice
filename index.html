Itemized List with Different Font Sizes

  • 12px - Absolute size suitable for mobile devices.
  • 16px - Default browser size, good for body text on web.
  • 2em - Relative size, twice the parent element's size.
  • 1.5rem - Relative to the root element, often used for headings.
  • 150% - Relative to the parent's font size.
  • 10pt - Absolute size, often used in print.
